コンセプト：cURL を設定する

このトピックでは、API リクエストのテストと作成に役立つツール cURL について学習します。cURL は HTTP リクエストを作成するコマンドラインツールです。ブライトコーブ API 呼び出しをテストするために、シェルスクリプトで cURL を使用します。Mac macOS または Unix/Linux システムを使用している場合は、すでに cURL があります。Windows の場合は、cURL を取得する方法については、以下の手順を参照してください。

ウィンドウズ

Windowsをお持ちの場合、cURLは以下のようにダウンロードしてインストールできます。

cURL をインストールする

  1. のウィンドウズ、という新しいフォルダを作成しますカールあなたのC：ドライブで。

    C:curl

  2. https://cygwin.com/install.htmlの指示に従って、使用しているシステムに応じて 32 ビットまたは 64 ビットのインストーラーを選択します。インストール中に、インストールするパッケージを要求するプロンプトが表示されます。Net パッケージを開き、 curlオプションを確認してインストールを完了します。

セットアップをテストする

cURLコマンドを実行するには、Terminal または DOS プロンプトにcurl (実行する要求に適切なフラグを付けて) と入力し、返す。cURL が正しくインストールされていることを確認するには、ターミナルウィンドウまたは DOS ウィンドウを開き、curlプロンプトで入力して Return キーを押します。次のようなメッセージが表示されるはずです。

 curl: try 'curl --help' or 'curl --manual' for more information

データを送信する

（POST リクエストや PATCH リクエストなど）API リクエストを使用してデータを送信する必要がある場合は、いくつかの方法があります。データは、インラインまたは別のファイルに含めることができます。データを別の JSON ファイルとして送信するプレーヤー管理 API の例を次に示します。

 curl --request POST
    --url https://players.api.brightcove.com/v2/accounts/57838016001/players
    --header 'authorization: Bearer YOUR_ACCESS_TOKEN_HERE'
    --header 'content-type: application/json'
    --data <player_data/player_config.json
ファイルの内容を送信する < "" 記号に注意してください。ファイルをアップロードとして添付する「@」記号ではなく、これを使用します。
