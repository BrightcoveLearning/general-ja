Ooyala ビデオクラウド取り込みアダプタによるリモートアセットの取り込み:MRSS によるリモートアセットの取り込み

MRSS を使用して、Video Cloud でリモートアセットを作成および更新できます。
注:これらのトピックは、Video Cloud に移行した Ooyala のお客様にのみ適用されます。

はじめに

メディア RSS (MRSS) を使用して、リモートアセットを追加または更新できます。

見るリモートアセットの操作 VideoCloudのリモートアセットをよりよく理解するため。

MRSS を使用してリモートアセットを追加または更新する場合、(代わりに < ooyala: remoteSet>)タグを指定して、アセットをリモートとして識別します。 < media: content> ) を MRSS フィード項目に表示します。

注:リモートアセットを追加するか更新するかを決定するために、Brightcoveは<ooyala:embedcode></ooyala:embedcode>のフィールドXMLマニフェストリモートアセットが現在存在するか（更新）、存在しないか（追加）を判別します。

リモートアセットの追加

次のサンプルコードは、リモートアセットを作成します (指定された < guid> は存在しません)。

リモートアセットの更新

既存のリモートアセットの場合、MRSS を使用して次のフィールドを更新できます。

  • メディア：タイトル
  • メディア：説明
  • ooyala：subTitle
  • ooyala：メタデータ
  • dcterms：valid
  • メディア：サムネイル
  • ooyala：labels

MRSSフィードアイテムを送信するときに、これらのフィールドのいずれかに更新が含まれている場合 <アイテム > タグを付けると、システムはリモートアセットを適切な変更で更新します。
