ここに挙げたトピックでは、Brightcove が Video Cloud への移行中に Ooyala の顧客を支援するために構築したコネクタとアダプタについて説明します。

注:これらのトピックは、Video Cloud に移行した Ooyala のお客様にのみ適用されます。

Brightcove は Ooyala から Video Cloud への移行に役立つ追加のツールに取り組んでいるため、定期的にここをチェックしてください。

  • Ooyala プレーヤーを使用してビデオクラウドコンテンツを配信する
  • ブライトコーブ DNS サーバーの追加
  • バックロット/ビデオクラウド CMS コネクタ
  • バックロットとビデオクラウド:ビデオとプレイリストフィールド
  • バックロット API リファレンス (ブライトコーブのバージョン)
  • ビデオクラウドへの移行:取り込みアダプタ
    • ウーヤラ-ビデオクラウド取り込みアダプタ:OIS2.5で摂取する
    • ウーヤラ-ビデオクラウド取り込みアダプタ:マニフェストファイル形式
    • ウーヤラ-ビデオクラウド取り込みアダプタ:入力ファイル形式
    • ウーヤラ-ビデオクラウド取り込みアダプタ:Aspera を使用した取り込み
    • ウーヤラ-ビデオクラウド取り込みアダプタ:SFTP を使用した取り込み
    • ウーヤラ-ビデオクラウド取り込みアダプタ:MRSS を使用した取り込み
    • ウーヤラ-ビデオクラウド取り込みアダプタ:MRSS を使用してリモートアセットを作成する
    • ウーヤラ-ビデオクラウド取り込みアダプタ:CSV マニフェストファイル
    • ウーヤラ-ビデオクラウド取り込みアダプタ:XML マニフェストファイル
  • ビデオクラウドへの移行:IQ API とビデオクラウド分析 API
  • ビデオクラウドプラットフォーム API の認証
